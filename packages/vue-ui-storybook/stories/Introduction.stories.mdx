<h1>关于CSDN DESIGN</h1>
<p>CSDN DESIGN是一套基于vue.js的开源组件库，主要服务于PC界面的中后台产品。</p>
<p>当前版本为 Version 0.1.0，后续将在 <a href="https://gitlab.csdn.net/fe/csfe/csdn-design" target="_blank">
  https://gitlab.csdn.net/fe/csfe/csdn-design </a>仓库继续开发与维护后续版本。
</p>

<p>开发团队：CSDN长沙前端研发团队</p>

<p>官网：<a href="design.csdn.net" target="_blank">design.csdn.net</a></p>

<h4>特性</h4>
<ul>
  <li>丰富的组件和功能，满足绝大部分网站场景</li>
  <li>提供专业、优质的一对一技术支持</li>
  <li>提供页面元素和模块化的概念</li>
  <li>符合大众审美的UI</li>
  <li>事无巨细的文档</li>
</ul>

<h4>安装</h4>
<p>使用npm</p>
<code>$ npm install csdn-design --save  </code>
