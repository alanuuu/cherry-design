# CSDN Design
<p>CSDN Design是一套基于Vue2.x的UI组件库。它依托CSDN丰富的项目和开发经验，沉淀出日常所用到的组件。</p>

<p>开发团队：CSDN长沙前端研发团队😊</p>

<p>地址：<a href="http://design.csdn.net" target="_blank">design.csdn.net</a></p>

<h3>特性</h3>
<ul>
  <li><strong>简洁、大方、美观、易用</strong></li>
  <li>开箱即用的 <strong>高质量</strong> Vue组件</li>
  <li>丰富的组件和功能，满足 <strong>绝大部分</strong> 网站场景</li>
  <li>充满 <strong>互动性</strong> 的文档，用户可 <strong>直接</strong> 操作组件样式</li>
</ul>

<h3>安装</h3>
<p>使用npm</p>

```shell
npm install @csdn-design/vue-ui --save
```

<p>使用yarn</p>

```shell
yarn add @csdn-design/vue-ui
```

<p><strong>目前只能使用该方式</strong> 内测版; </p>

```
npm install @csdn-design/vue-ui --save --registry=https://repo.csdn.net/repository/npm-group/
```
