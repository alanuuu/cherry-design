# Card 卡片

## Attributes
                                                              
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|:----|:----|:----:|:----:|:----:|
| shadow | 卡片阴影显示的方式 | string | always/hover/never | always |
| header | Card 的标题，也可通过slot（见下表）传入 | string | - | - |

## Slot

| name | 说明 |
|:----|:----|
| - | Card内容 |
| header | Card 标题区的内容 |
