# empty 空状态

## Attributes
                                                              
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|:----|:----|:----:|:----:|:----:|
| image | 图片地址，图片也可通过slot（见下表）传入 | string | - | cherry默认图片 |
| description | 描述，也可通过slot（见下表）传入 | string | - | - |
| imageSize | 图片大小 | number | - | 200 |

## Slot

| name | 说明 |
|:----|:----|
| image | 图片 |
| description | Empty 描述区的内容 |
| bottom | Empty 底部的内容 |
